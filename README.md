## About 
This project is a minimum viable integration of Google Pay for Web using HTML and JavaScript.

## Learning
To learn about the code in this project template, follow to the Google Pay API for Web 101: Basic codelab. Learning paths help you get the most of your integration by taking you through a guided developer journey from start to finish.

https://developers.google.com/learn/pathways/google-pay-for-web?hl=en

## Create a merchant account
While a merchant ID isn't required for testing your integration in a TEST enviornment, you will need one when you deploy to a PRODUCTION environment. Register with the Google Pay & Wallet Console to receive your merchant ID. It's quick and easy! Get yours now.

## Important Note
This project is based on the official Google Learn resources. The code and implementation details follow Google's recommended guidelines for integrating Google Pay on the web.
